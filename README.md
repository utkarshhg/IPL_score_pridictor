# 🏏 IPL Score Predictor

This is a simple Machine Learning project where I built a web app to predict the final score of an IPL match based on the current match situation.

The idea is to take inputs like current runs, overs, wickets, teams, and venue, and estimate what the final score could be.

---

## 🚀 What this project does

* Takes live match inputs
* Uses a trained ML model
* Predicts the final score instantly
* Displays the result through a simple web interface

---

## 🧠 Tech used

* Python
* Pandas
* Scikit-learn
* Streamlit

## ▶️ How to run this project

1. Clone the repository

```
git clone https://github.com/your-username/ipl-predictor.git
cd ipl-predictor
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the app

```
streamlit run app.py
```

## 📊 Input fields

The model uses:

* Batting team
* Bowling team
* City
* Current runs
* Wickets fallen
* Overs completed

## ⚠️ Note

Make sure the scikit-learn version is the same as the one used while training the model, otherwise the model may not load properly.
